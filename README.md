# Basic Web Starter

This repository contains a minimal HTML/CSS/JavaScript starter for web projects.

## Files

- `index.html` – main HTML document
- `style.css` – basic styling
- `script.js` – starter JavaScript file

## Usage

1. Open `index.html` in your web browser.
2. Modify the files as needed for your project.

This README is configured for GitHub and displays automatically on the repository's main page.
